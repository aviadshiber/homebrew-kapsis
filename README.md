# Homebrew Tap for Kapsis

This is the official [Homebrew](https://brew.sh/) tap for [Kapsis](https://github.com/aviadshiber/kapsis) - Hermetically Isolated AI Agent Sandbox.

## Installation

```bash
brew install aviadshiber/kapsis/kapsis
```

Or tap first, then install:

```bash
brew tap aviadshiber/kapsis
brew install kapsis
```

## Upgrading

```bash
brew upgrade kapsis
```

## Uninstalling

```bash
brew uninstall kapsis
brew untap aviadshiber/kapsis
```

## What is Kapsis?

Kapsis enables running multiple AI coding agents (Claude Code, Codex CLI, Aider) in parallel on the same project with complete isolation. Each agent runs in a Podman container with Copy-on-Write filesystem.

Learn more: [github.com/aviadshiber/kapsis](https://github.com/aviadshiber/kapsis)

## Automatic Updates

This formula is automatically updated when new versions of Kapsis are released.
